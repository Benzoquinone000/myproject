分层异步架构

基于 FastAPI + LangGraph 的单体分层架构，全链路 async/await，SSE 流式推送，中间件链实现鉴权、限流、日志等横切关注点。

RAG + 图谱增强

Milvus 向量检索 + LightRAG 图增强双引擎，结合 Neo4j 知识图谱，支持 40+ 文档格式解析与多种 OCR 引擎。

异构多模态存储

SQLite 关系数据 + MinIO 对象存储 + Milvus 向量索引 + Neo4j 图数据库 + 文件系统，按数据特征选择最优存储引擎。
