# 金析智策（JinXiZhiCe）使用说明文档

## 1. 文档目的

本文档用于指导评审专家、指导教师与项目成员快速体验和使用金析智策系统，包括登录、核心功能操作、部署访问与常见问题处理。

---

## 2. 系统简介

金析智策是一个面向金融场景的智能分析与决策支持平台，采用多智能体协作架构，融合知识库检索（RAG）、知识图谱、评测看板与工具调用能力，支持从“文档入库”到“智能问答与分析”的完整流程。

## 4. 账号与权限说明

系统采用基于角色的访问控制（RBAC）：

- `superadmin`：系统最高权限（初始化创建）
- `admin`：管理权限（知识库、图谱、评测、看板、系统配置）
- `user`：普通用户（对话与基本使用）

首次部署时，可通过初始化接口创建管理员账户；正式答辩演示建议使用管理员账号。

---

## 5. 快速上手流程（推荐演示顺序）

1. 登录系统，进入首页。  
2. 打开知识库页面，创建知识库并上传文档。  
3. 等待系统自动完成解析、分块、向量化与入库。  
4. 进入智能体对话页面，选择金融分析智能体并提问。  
5. 查看工具调用结果与参考片段，验证检索与回答质量。  
6. 打开知识图谱页面，执行子图查询与实体关系浏览。  
7. 打开数据看板查看系统统计和调用分析。  
8. 在评测模块上传或生成基准集，执行RAG评测并查看指标结果。

## 6. 核心功能使用说明

## 6.1 多智能体对话

- 入口：对话页面（智能体选择区）
- 支持智能体：
  - 通用对话智能体
  - 深度推理智能体
  - 金融分析智能体
- 说明：回答以流式方式返回，可查看工具调用中间结果。

## 6.2 知识库管理

- 入口：知识库页面
- 支持能力：
  - 新建/编辑/删除知识库
  - 上传文档并自动处理（解析、分块、索引）
  - 检索与导出
- 支持类型：Milvus 向量检索、LightRAG 图增强检索
- 支持格式：40+ 文档类型（如 PDF、Word、PPT、Markdown、图片等）

## 6.3 知识图谱

- 入口：图谱页面
- 功能：
  - 子图查询（关键词、深度、节点数限制）
  - 实体与关系可视化
  - 实体导入/删除与统计查看

## 6.4 RAG评测

- 入口：评测页面
- 功能：
  - 上传基准数据集（JSONL）
  - 自动生成基准集
  - 执行评测并查看历史结果
- 常见指标：
  - 召回率（Recall）
  - 准确率（Precision）
  - 答案质量评分

## 6.5 数据看板

- 入口：Dashboard 页面
- 内容：
  - 用户活跃度
  - 智能体使用统计
  - 工具调用排行与成功率
  - 知识库状态与系统调用时序

## 7. 部署与运行说明（简版）

### 7.1 Docker 一键启动

在项目根目录执行：

```bash
docker compose up -d
```

如需 OCR 扩展服务（GPU）：

```bash
docker compose --profile all up -d
```

### 7.2 关键依赖服务

- API（FastAPI）
- Web（Vue 3）
- Milvus（向量数据库）
- Neo4j（图数据库）
- MinIO（对象存储）
- etcd（Milvus元数据）

## 8. 常见问题（FAQ）

1. **登录失败或无权限**  
   请确认账号角色是否为 `admin/superadmin`，并检查Token是否过期。

2. **上传后文件长时间处理中**  
   大文件解析与向量化耗时较长，可在任务中心查看进度，或检查OCR服务健康状态。

3. **图谱页面无数据**  
   请先确认文档已完成入库，并检查图谱构建来源配置是否正确。

4. **回答质量波动**  
   建议调整知识库类型、分块参数、Embedding/Reranker模型，并通过评测模块对比结果。

5. **在线地址不可访问**  
   检查域名解析、反向代理、HTTPS证书与后端服务健康状态。
