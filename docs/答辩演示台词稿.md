## 开场（约 20 秒）

各位老师好，今天向大家汇报我们的设计项目——**金析智策**。

金析智策是一个**智能体赋能的金融数据分析与决策支持系统**，核心思路是将大语言模型的推理能力与知识库检索、知识图谱相结合，为用户提供智能化的金融分析和决策辅助。

## 一、项目背景与目标（约 30 秒）

在金融领域，分析师日常面对大量的研报、公告、财务数据，传统的信息检索方式效率较低，且难以进行跨文档的关联分析。

本项目的目标是构建一个**多智能体协作平台**，让用户可以：
- 通过**自然语言对话**获取金融分析结论
- 利用**知识库**管理和检索专业文档
- 借助**知识图谱**发现实体间的隐含关系
- 通过**数据看板**监控系统运行状态

## 二、技术架构（约 50 秒）

系统采用前后端分离的分层架构，我从下往上简要介绍。

**数据层**，我们使用了五种存储引擎各司其职：SQLite 存储用户和会话数据，Milvus 作为向量数据库支撑语义检索，Neo4j 存储知识图谱的实体和关系，MinIO 负责文件对象存储，以及本地文件系统保存配置和日志。

**领域服务层**是系统的核心，分为三大模块：
- **智能体模块**，基于 LangGraph 状态机实现，内置三种智能体——通用对话、深度推理和金融分析，支持工具调用、MCP 协议扩展和人工审批中断
- **知识库模块**，采用工厂模式，同时支持 Milvus 向量检索和 LightRAG 图增强检索两种方式
- **支撑服务**，包括对话管理、异步任务队列、内容审核、RAG 评测等

**接入层**基于 FastAPI 构建。

**前端**使用 Vue 3 + Ant Design Vue。

整个系统通过 Docker Compose 编排，一键即可部署全部服务。

## 三、核心功能演示（约 90 秒）

接下来我演示几个核心功能。

### 3.1 多智能体对话

【画面：打开对话页面，选择金融分析智能体】

这是对话界面，左侧是历史会话列表，右侧是聊天区域。我选择金融分析智能体，输入一个问题。

可以看到系统以 SSE 流式方式实时返回回答。在回答过程中，智能体自动调用了知识库检索工具，从我们预先导入的文档中找到了相关内容，并基于检索结果生成了分析结论。

这里展示的工具调用结果是可展开的，可以看到具体检索到了哪些文档片段。

### 3.2 知识库管理

【画面：切换到知识库页面】

知识库支持 40 多种文件格式的导入。上传文件后，系统会自动进行文档解析、分块、向量化和入库。

我们支持两种知识库类型：Milvus 向量检索适合精确语义匹配，LightRAG 图增强检索则能利用知识图谱的结构化信息提升检索质量。

### 3.3 知识图谱可视化

【画面：切换到知识图谱页面】

这是知识图谱的可视化界面，可以看到从文档中自动抽取的实体和关系。支持按关键词搜索子图，点击节点可以查看详细信息。

### 3.4 数据看板

【画面：切换到 Dashboard 页面】

数据看板提供了多维度的统计分析，包括用户活跃度、智能体使用情况、工具调用排行、知识库状态等，支持时序数据的多粒度展示。

## 四、技术亮点（约 30 秒）

总结几个技术亮点：

第一，**LangGraph 多智能体架构**，通过状态机实现灵活的对话流程控制，支持工具调用和人工审批中断。

第二，**双引擎知识检索**，向量检索与图增强检索互补，提升了检索的准确性和召回率。

第三，**可扩展的插件体系**，文档处理支持多种 OCR 引擎，工具系统支持 MCP 协议扩展，智能体支持自动发现和热重载。

第四，**完整的工程化实践**，包括 JWT 鉴权、登录限流、内容审核、RAG 评测、Docker 一键部署等。
